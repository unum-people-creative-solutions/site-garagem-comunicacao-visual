# Tech Stack - Garagem LP

## Core
- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 (Oxide engine)
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Integrations
- **CRM**: Unum People API (/ingest)
- **Analytics**: Google Tag Manager / gtag.js

## Infrastructure
- **Hosting**: Vercel (preferencial)
- **SEO**: Next.js Metadata API
