# Feature Spec: Content & Services (Grid, Workflow, FAQ)

## User Story
Como cliente em potencial, quero entender quais serviços a Garagem oferece e como funciona o processo de contratação para que eu possa tomar uma decisão informada.

## Requirements
- **FR1 (Services Grid)**: Exibir os 3 pilares de serviço (Fachadas, Envelopamento, Sinalização) com descrições e ícones.
- **FR2 (Workflow)**: Apresentar o fluxo de 4 passos (Orçamento, Visita, Design, Produção) de forma visual.
- **FR3 (FAQ)**: Implementar componente de acordeão para as 6 perguntas frequentes.
- **FR4 (Footer)**: Exibir dados de contato, endereço em Colombo-PR e links sociais.
- **FR5 (Responsive)**: Grade de serviços deve colapsar para 1 coluna em mobile.

## Acceptance Criteria
- [ ] O componente de FAQ permite abrir/fechar as respostas.
- [ ] O fluxo de trabalho é intuitivo e segue a ordem lógica.
- [ ] Links no footer funcionam corretamente (WhatsApp, Endereço).

## Design References
- Ícones: Lucide React (espesura grossa).
- Estilo: Blocos com recortes angulares e bordas Amarelas.
